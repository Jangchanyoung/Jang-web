HTML이란 무엇인가?

HTML은 문서의 글자크기, 글자색, 글자모양, 그래픽, 문서이동(하이퍼링크) 등을 정의하는 명령어로서 홈페이지를 작성하는 데 쓰인다.


The H1 tag is an HTML element used to define the main heading of a webpage. H1 tag is supported by all the major browsers such as Chrome, Android, Firefox, Internet Explorer (IE), Opera, Safari, and others. There are around 6 headings that define the overall content structure in a webpage.
